### **Functional Programming using JS**

**Topics**  
  ** OOPS-1: this Keyword**

* Execution context & binding rules  
* Global vs function vs object context  
* this in strict mode vs non-strict mode  
* Arrow functions vs regular functions  
* Implicit, explicit, and default binding

 **OOPS-2: Constructor Functions & new Keyword**

* Constructor functions  
* Internal working of new  
* Object creation flow (this, prototype linking)  
* Manual implementation of new (polyfill concept)

**OOPS-3: Prototypes & Prototype Chain**

* prototype vs \_\_proto\_\_  
* Prototype chaining & inheritance  
* Method lookup & shadowing  
* hasOwnProperty, in, instanceof  
* Function prototypes vs object prototypes

**OOPS-4: Classes & Inheritance in JS**

* ES6 Classes vs Constructor functions  
* extends and super  
* Method overriding  
* Static methods & private fields  
* Reality of OOP in JavaScript (prototype-based nature)

**Module 2: Function Mastery & Functional JS**

**Call, Apply & Bind**

* Explicit binding  
* Differences between call, apply, bind  
* Polyfills for all three methods  
* Edge cases with this

**Higher-Order Functions & Array Polyfills**

* Concept of HOFs  
* Native methods: map, filter, reduce, forEach  
* Writing polyfills for array methods  
* Functional programming patterns

**Closures & Scope**

* Lexical scope  
* Closure use cases  
* Data encapsulation  
* Currying & partial application  
* Function factories

**Module 3: Asynchronous JavaScript**

**Promise Internals**

* Promise states & lifecycle  
* Chaining & resolution  
* Microtasks vs macrotasks  
* Writing Promise polyfill

**Event Loop & Runtime**

* Call stack  
* Web APIs / Node APIs  
* Task queue vs microtask queue  
* Event loop execution flow

**Async Patterns & Concurrency**

* Sequential vs parallel execution  
* Promise combinators (all, race, any, allSettled)  
* Concurrency control (pooling, batching)  
* Real-world async workflows

**Async Error Handling**

* try...catch with async/await  
* Error propagation in promises  
* Retry mechanisms  
* Graceful failure handling

**Module 4: Performance & Optimization**

**Debouncing & Throttling**

* Differences & use cases  
* Implementation from scratch  
* Real-world scenarios (search, scroll, resize)

**Memory Management & Garbage Collection**

* Stack vs heap  
* Garbage collection basics  
* Memory leaks (closures, DOM references, timers)  
* Optimization strategies

**Cloning, Equality & Immutability**

* Shallow vs deep copy  
* Object.assign, spread operator  
* Deep cloning techniques  
* Value vs reference comparison  
* Immutable patterns

**Memoization & Caching**

* Concept & benefits  
* Implementing memoization  
* Cache invalidation basics  
* Performance optimization patterns

**JS Weird Parts (Advanced Edge Cases)**

* Type coercion quirks  
* Hoisting behavior  
* \== vs \===  
* NaN, undefined, null differences  
* Weird interview edge cases
